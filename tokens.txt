indent: represents the number of spaces used to indent the current line
outdent: represents the number of spaces used to unindent the current line
newline: represents a newline character
tag: represents an HTML tag, such as html, head, body, div, etc.
id: represents an HTML id attribute, such as #my-id
class: represents an HTML class attribute, such as .my-class
attribute: represents an HTML attribute, such as src="/path/to/file.js"
text: represents text content within an HTML tag, such as Hello World!
interpolated-code: represents PUG code that is evaluated and inserted into the HTML output, such as #{myVariable}
comment: represents an HTML comment, such as <!-- This is a comment -->
doctype: represents the HTML doctype declaration, such as doctype html
